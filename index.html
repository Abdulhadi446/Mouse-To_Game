Distribution Hub

Mouse-To_Game Download and Setup

This page is the root landing website for distributing the project. Users can play the web demo, download pure executables, or build local binaries for Linux and Windows.

Open Web Demo GitHub Repository Download Artifacts

Windows

Recommended: Command Prompt build

PowerShell blocked? Use the CMD script to generate standalone .exe files.

build_windows.cmd

Outputs:

  • dist/mouse-to-game.exe
  • dist/input-remapper-runner.exe

Linux

One-file executable build

Build native Linux binaries with PyInstaller in one command.

./build_linux.sh

Outputs:

  • dist/mouse-to-game
  • dist/input-remapper-runner

Direct Download

No local build needed

Use GitHub Actions artifacts to grab prebuilt Linux and Windows binaries.

Actions -> Build Executables -> Artifacts

Look for:

  • executables-Linux
  • executables-Windows

Quick Distribution Steps

1. Push updates to main or create a tag like v1.0.0.

2. Run the Build Executables GitHub workflow.

3. Share downloaded binaries or upload them to a release.

4. Point users to this root page for setup links and demo access.

Note: Linux evdev and uinput runtime features may still require device permissions on end-user systems.
